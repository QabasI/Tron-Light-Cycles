# LightCycles Game

## Overview

LightCycles is a Java-based game inspired by the classic Tron light cycle game. The game features both single-player and two-player modes, allowing players to control bikes that leave trails on a grid. The objective is to avoid crashing into walls, your own trail, or your opponent's trail, while trying to outlast the other player.

## Features

- Single-player mode against an AI opponent
- Two-player mode on the same keyboard
- Simple and intuitive controls
- Retro graphics and sound effects

## Requirements

- Java Development Kit (JDK) 8 or higher
- A computer with a keyboard and display

## Installation

1. Clone the repository to your local machine:
    ```sh
    git clone https://github.com/yourusername/lightcycles.git
    ```
2. Navigate to the project directory:
    ```sh
    cd lightcycles
    ```

## Running the Game

1. Compile the Java source files:
    ```sh
    javac *.java
    ```
2. Run the game:
    ```sh
    java LightCycles
    ```

## Controls

### Player 1
- **Up:** W
- **Down:** S
- **Left:** A
- **Right:** D

### Player 2
- **Up:** Up Arrow
- **Down:** Down Arrow
- **Left:** Left Arrow
- **Right:** Right Arrow

## Game Rules

- Players control their bikes using the specified keys.
- Each bike leaves a trail behind as it moves.
- The goal is to avoid crashing into walls, your own trail, or your opponent's trail.
- The last player remaining wins the game.
